# 8-Bit Arithmetic Logic Unit (ALU)

## Project Overview
This project implements an 8-bit Arithmetic Logic Unit (ALU) using Verilog HDL in Xilinx Vivado. The ALU performs various arithmetic and logical operations based on the control signal.

## Objective
The objective of this project is to design, simulate, and verify an 8-bit ALU capable of performing multiple arithmetic and logical operations using Verilog HDL.

## Features
- 8-bit Addition
- 8-bit Subtraction
- Bitwise AND
- Bitwise OR
- Bitwise XOR
- Bitwise NOT
- Left Shift
- Right Shift

## Tools Used
- Verilog HDL
- Xilinx Vivado

## Project Files
- **alu.v** – Main ALU design
- **tb_alu.v** – Testbench used for simulation

## Applications
- Microprocessors
- Digital Signal Processing
- Embedded Systems
- FPGA-Based Digital Design
